# Zen MCP Go Backend

<Callout type="info" emoji="ℹ️">
  This is the documentation for the Go implementation of the Zen Model Context Protocol (MCP) server.
</Callout>

## Getting Started

```bash
# Clone the repository
git clone https://github.com/WesHacixo/zen-mcp-server.git
cd zen-mcp-go

# Build and run
go build -o bin/server ./cmd/server
./bin/server
```
